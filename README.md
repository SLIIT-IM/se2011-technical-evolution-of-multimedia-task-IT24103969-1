### Processing Programming with Creative Coding

---

## 1. Assignment Title
**Creative Visual Storytelling Using Processing**

---

## 2. Assignment Purpose (Why this assignment?)
This assignment assesses your ability to apply Processing concepts learned during **Lab 01–04** to create a meaningful and creative visual artwork.

The focus is **not** on complexity, but on:
- Clear understanding of fundamentals  
- Creative expression  
- Logical use of code to produce visuals  

---

## 3. Assignment Objective
To design and implement a simple **interactive and/or animated** creative artwork using Processing, demonstrating understanding of **shapes, colors, logic, and animation principles**.

---

## 4. Learning Outcomes Addressed (Module Alignment)
This assignment assesses the following module learning outcomes:
- **LO1:** Analyze how technical choices influence visual design  
- **LO3:** Apply technical knowledge to create a multimedia artifact  
- **LO4:** Analyze the relationship between technical decisions and aesthetic outcome  

---

## 5. Assignment Description
Create a creative visual artwork using **Processing (Java mode)**.

Your artwork must:
- Use basic shapes (`ellipse`, `rect`, `line`, etc.)
- Include **motion and/or interaction**
- Communicate a simple **idea, emotion, or message**

 This is a creative assignment, but it must be built only using concepts taught in **Labs 01–04**.

---

## 6. Theme Selection (Choose ONE)
Choose **one** theme from the list below:

1. **Emotion**  
   *(Happy, calm, stress, loneliness, excitement)*  
2. **Daily Life**  
   *(Traffic, classroom, weather, city movement)*  
3. **Nature**  
   *(Rain, waves, wind, clouds, sunrise)*  
4. **Abstract Art**  
   *(Patterns, rhythm, repetition, motion)*  

 The theme must be clearly visible in the animation.

---

## 7. Technical Requirements (Mandatory)

### A. Visual Elements
Your sketch MUST include:
- Minimum **5 shapes**
- Minimum **3 different colors**
- Proper use of **canvas size**

### B. Programming Concepts
Your sketch MUST include:
- At least **2 variables**
- At least **1 conditional statement** (`if / else`)
- At least **1 loop** (`for`)
- Use of `draw()` for animation

### C. Motion / Interaction (Choose at least ONE)
- Automatic animation (moving objects)  
- Mouse interaction (`mouseX` / `mouseY`)  
- Combination of both

## 9. Step-by-Step Guidance (For Students)

### Step 1: Planning (Before Coding)
- Decide your theme  
- Sketch your idea on paper  
- Identify:
  - What moves?
  - What stays static?
  - What interaction exists?

### Step 2: Base Setup
- Create canvas using `size()`  
- Set background color  

### Step 3: Add Shapes
- Draw shapes using coordinates  
- Position elements clearly  

### Step 4: Add Motion / Interaction
- Use variables to move objects  
- Apply `if` for control  
- Use `for` loop for repetition  

### Step 5: Improve Aesthetics
- Adjust colors  
- Improve spacing  
- Control speed  

### Step 6: Final Testing
- Ensure sketch runs smoothly  
- Ensure the idea/theme is visible
